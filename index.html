Screen Details:

The screen above shows the distinct elements needed as part of the user interface. The left pane contains an input box and the "Get Money"
button apart from the heading. The content for the right pane is dynamically generated based on the computed values.

Core Functionality (Workflow)

The user enters the amount (whole numbers only) needed in the text field. On clicking "Get Money," the app computes the least number of notes needed to dispense
the exact amount. The possible denominations are Rs 1, 2, 5, 10, 20, 50, 100, 200, and 2000. The computed count is displayed on the right panel, as shown in the image above.
